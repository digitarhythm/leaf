機能ガイド

「管理」を自動化し、「執筆」に集中するための機能解説

1. メインメニュー（画面左上）

画面左上のメニューバーには、ドキュメントのライフサイクルを管理するための3つの重要なアイコンが並んでいます。

[Screenshot: Menu Bar Icons Placeholder]
🆕

新規シート作成 Opt/Alt + n

Googleドライブ上に新しいテキストファイルを作成し、即座に編集を開始します。ファイル名を決める必要はありません。Leafが自動的に「Untitled」として管理を開始します。

🗂️

編集シート選択ダイアログ Opt/Alt + m

Leafの「管理画面」です。カテゴリー別に整理されたドキュメントを、内容のプレビューを見ながら視覚的に探せます。ファイル名よりも「中身」で判断するLeaf特有の管理スタイルです。

📂

ローカルファイル読み込み Opt/Alt + o

PC内の既存ファイルを開いて編集します。このモードでは元のファイルのエンコード（BOMの有無など）が維持されます。カテゴリーを変更してGoogleドライブへ移動させると、自動的にクラウド同期対象となります。

2. 執筆を支える機能（ステータスバー）

画面下部のステータスバーでは、エディタの挙動をリアルタイムにカスタマイズできます。

Vimモード切り替え

[Screenshot: Vim Toggle Button Placeholder]

右端の「Vim」ボタンで、プロフェッショナルなVimキーバインドを有効化できます。H/J/K/Lによる移動や、強力なコマンド操作がブラウザ上でそのまま利用可能です。設定はブラウザに保存され、次回起動時も維持されます。

拡張子変更とプレビュー

[Screenshot: Extension Badge Placeholder]

中央の拡張子バッジをクリックすると、ファイルの形式を変更できます。これに連動して以下の機能が切り替わります：

  • シンタックスハイライト: rs js など、言語に応じた色付けが行われます。
  • プレビュー表示: Opt/Alt + l を押した際、md なら整形されたドキュメント、ソースコードならハイライト済みのコードが表示されます。

3. ネットワークと自動保存

Leafは「保存」という概念を意識させない設計になっています。

オフライン・オンラインの自動切り替え

[Screenshot: Network Status Indicator Placeholder]

左下のネットワークインジケータが現在の状態を示します。

  • オンライン時: 内容を変更するたびにGoogleドライブへ自動保存されます。
  • オフライン時: ネットワークが切断されると、即座にブラウザ内の安全な領域（IndexedDB）への保存に切り替わります。作業を止める必要はありません。
  • オンライン復帰時: 再接続されると、Leafはバックグラウンドで「サイレント同期」を実行します。オフライン中の編集内容が自動的にGoogleドライブへ反映されます。
