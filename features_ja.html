特徴と機能ガイド

「管理」を自動化し、「書くこと」に集中する

1. メインメニュー（左上）

左上のメニューバーには、ドキュメントのライフサイクルを管理するための重要な3つのアイコンが並んでいます。

[Screenshot: Menu Bar Icons Placeholder]
🆕

新規シート Opt/Alt + n

Googleドライブ上に新しいテキストファイルを作成し、瞬時に書き始めることができます。ファイル名を考える必要はありません。Leafが背後で「Untitled」として自動管理します。

🗂️

シート選択ダイアログ Opt/Alt + m

Leafの「管理画面」です。カテゴリごとに整理されたドキュメントを、中身をプレビューしながら視覚的に探すことができます。ファイル名ではなく「中身」で判断する、Leafならではのモダンなドキュメント管理です。

📂

ローカルファイルの読み込み Opt/Alt + o

PC上にある既存のファイルを開いて編集します。このモードでは元ファイルのエンコーディング（BOM等）が維持されます。カテゴリを変更してGoogleドライブへ移動させることで、自動的にクラウド同期の対象に切り替わります。

2. 執筆をサポートする機能（ステータスバー）

下部のステータスバーから、エディタの挙動をリアルタイムにカスタマイズできます。

Vimモードの切り替え

[Screenshot: Vim Toggle Button Placeholder]

右端の「Vim」ボタンで、プロフェッショナル向けのVimキーバインドを有効にできます。H/J/K/Lでの直感的な移動や、強力なコマンド操作をブラウザ上でそのまま利用できす。選択状態はブラウザに保存され、次回の訪問時にも引き継がれます。

拡張子とプレビューの変更

[Screenshot: Extension Badge Placeholder]

中央の拡張子バッジをクリックすると、ファイルのフォーマットを変更できます。これは以下の2つを制御します。

  • シンタックスハイライト: rs や js など、言語に合わせた色分けが行われます。
  • プレビュー表示: Opt/Alt + l でプレビューをトグルできます。md 形式なら美しくレンダリングされたドキュメントが、ソースコードならハイライトされたコードブロックが表示されます。

3. ネットワークと自動保存

Leafでは、あなたが「保存」を意識する必要がないように設計されています。

オフライン/オンラインの自動切り替え

[Screenshot: Network Status Indicator Placeholder]

左下のネットワークインジケーターが現在の状態を示します。

  • オンライン時: 文字を打ち込むたびに、Googleドライブへ自動的かつリアルタイムに保存されます。
  • オフライン時: ネットワークが切断されると、瞬時にブラウザ内部の安全な領域（IndexedDB）への保存に切り替わります。作業を止める必要はありません。
  • 再接続時: オンラインに復帰すると、バックグラウンドで「サイレントシンク」が走り、オフライン時の編集内容を自動でGoogleドライブへ反映します。
