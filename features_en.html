Features Guide

Automate "Management" and focus on "Writing"

1. Main Menu (Top Left)

The top-left menu bar features three key icons for managing your document's lifecycle.

[Screenshot: Menu Bar Icons Placeholder]
🆕

New Sheet Opt/Alt + n

Instantly start writing by creating a new text file on Google Drive. No need to worry about filenames; Leaf automatically manages it as "Untitled".

🗂️

Sheet Selection Dialog Opt/Alt + m

Leaf's "Management Screen". This allows you to visually browse documents organized by category while previewing their content. It's Leaf's unique approach to document management—judging by "content" rather than just filenames.

📂

Load Local File Opt/Alt + o

Open and edit existing files on your PC. In this mode, the original file's encoding (e.g., BOM) is preserved. Changing its category and moving it to Google Drive will automatically enable cloud sync for it.

2. Features Supporting Your Writing (Status Bar)

You can customize the editor's behavior in real-time using the status bar at the bottom.

Toggle Vim Mode

[Screenshot: Vim Toggle Button Placeholder]

The "Vim" button on the far right enables professional Vim keybindings. Move around with H/J/K/L and utilize powerful command operations right in your browser. Your choice is saved in the browser and will persist on your next visit.

Change Extension and Preview

[Screenshot: Extension Badge Placeholder]

Clicking the extension badge in the center allows you to change the file format. This controls

  • Syntax Highlighting: Changes coloring to match the language such as rs or js.
  • Preview Display: Pressing Opt/Alt + l toggles the preview. md formats are elegantly rendered, while source code features highlighted code blocks.

3. Network and Auto-Save

Leaf is designed so you never have to think about "saving".

Automatic Offline/Online Switching

[Screenshot: Network Status Indicator Placeholder]

The network indicator in the bottom-left corner shows your current status.

  • When Online: Automatically saves to Google Drive with every change you make.
  • When Offline: Should your network drop, saving instantly switches to a secure area within your browser (IndexedDB). No need to stop working.
  • Upon Reconnecting: Once back online, Leaf performs a "Silent Sync" in the background, automatically uploading your offline edits to Google Drive.
