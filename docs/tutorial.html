Leaf

思考をファイル管理の束縛から解放する、次世代のメモ帳

デザインコンセプト：管理しない管理

Leafには「フォルダ階層」や「ファイル名に悩む時間」はありません。あなたの思考を即座に記録し、視覚的に管理するための仕組みを提供します。

主な特徴

  • Googleドライブ同期: 全てのシートはマイドライブ内の専用フォルダに保存され、どのデバイスからでも同じ環境で編集できます。
  • カテゴリー & 目視管理: ファイル名ではなく「カテゴリー」と「内容のプレビュー」で直感的にシートを探せます。 Opt/Alt + m で一覧を表示。
  • オートセーブ: 編集はリアルタイムで保存されます。「保存し忘れ」によるデータの損失を心配する必要はありません。
  • インテリジェント・プレビュー: Opt/Alt + l で、Markdownのレンダリングやソースコードのシンタックスハイライト（Tokyo Nightテーマ）を確認できます。
  • ローカルファイル対応: Opt/Alt + o でPC内のファイルを開けます。既存のエンコードを尊重しつつ、必要に応じてクラウドへ移行可能です。

クイックショートカット

キー 機能
Opt/Alt + n 新規シート作成（Googleドライブ）
Opt/Alt + Shift + n 新規ローカルファイル作成（BOM付きUTF-8）
Opt/Alt + m シート選択ダイアログを表示
Opt/Alt + l プレビュー（Markdown / Code）の切り替え
Opt/Alt + o ローカルファイルを開く
Opt/Alt + s 強制保存（通常は自動保存されます）

データ保護ロジック

Leafは、リロード時などの予期せぬトラブルからあなたのデータを守ります。エディタが異常にクリアされたことを検知すると、自動保存を一時停止し、直前の正しい状態を復元します。
