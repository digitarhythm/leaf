# テキストエディター「Usualy」要件定義

* アプリケーションはWebAssemblyで実装（言語はRust）
  * DOMの操作やブラウザ機能はJavaScriptライブラリを呼び出す形で使用
* テキスト編集は「vim」モードのオン／オフが出来る（デフォルトはオフ）
  * テキスト編集には「Ace Editor」ライブラリを使用
* エディタはタブ方式で複数開く事が出来る（ひとつのタブを「シート」と呼ぶ）
* 設定系のデータはlocalStorageに保存、データ系はIndexedDBを使用
  * IndexedDBは、DBの構造が変更された場合にバージョン番号を上げる
* ファイルはファイル名を付けて保存された後は、自動保存される
  * 自動保存は、最後に編集されてから5秒間なにも編集作業が行われなかった場合に、行われる
* 使用するにはGmailアカウントでログインが必要
  * ログイン時に取得する情報はメールアドレスと、GoogleドライブへのRead/Write権限
  * ログイン後にGoogleドライブに標準で存在する「マイドライブ」の下に「ApplicationSupport/UsualyData」というディレクトリを作成する
  * ファイルオープン／保存時にダイアログでデフォルトで開くディレクトリは「UsualData」になる
* アプリケーション起動直後は、テキストがなにも書かれていないまっさらなシートを開く
* 明示的に保存動作を行うまでは、テキストデータはIndexedDBにテンポラリファイル名を付けて保存する
* ショートカットは、Macはコマンドキー、Winはコントロールキーの動作を奪う
  * コマンド＋ｗ（Mac）、コントロール＋ｗ（Win）を押すとブラウザのタブが閉じるのではなく、エディタのタブが閉じる
* テキストファイルを読み込んで開いた時に、文字コードがBOM付きUTF-8ではなかった場合はBOM付きUTF-8に変換するかのダイアログを出す
  * 「OK」がクリックされた時は、以降そのファイルはBOM付きUTF-8として扱う（自動保存もBOM付きUTF-8で上書き保存する）
* シート画面最上段にボタンが並んだボタンバーを配置する
  * ボタンは左から「新規シート作成」「フロッピーディスクアイコン（保存）」「vimモードON/OFF」を配置
