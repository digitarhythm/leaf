功能指南

自动化"管理"，让您专注于"写作"

1. 主菜单 (左上角)

左上角的菜单栏有三个关键图标，用于管理您的文档生命周期。

[Screenshot: Menu Bar Icons Placeholder]
🆕

新建工作表 Opt/Alt + n

通过在 Google 云端硬盘上创建新的文本文件，立即开始写作。无需担心文件名；Leaf 会自动将其作为"无标题"管理。

🗂️

工作表选择对话框 Opt/Alt + m

Leaf 的"管理屏幕"。使您能在预览内容的同时，直观地浏览按类别组织的文档。这是 Leaf 独特的文档管理方式——通过"内容"而不是仅仅通过文件名来判断。

📂

加载本地文件 Opt/Alt + o

打开和编辑您电脑上已有的文件。在此模式下，原始文件的编码 (例如 BOM) 将被保留。将其类别更改并移动到 Google 云端硬盘后，将自动为其启用云同步。

2. 辅助写作功能 (状态栏)

您可以使用底部的状态栏实时定制编辑器的行为。

切换 Vim 模式

[Screenshot: Vim Toggle Button Placeholder]

最右侧的"Vim"按钮可启用专业的 Vim 快捷键绑定。使用 H/J/K/L 移动，并在浏览器中直接利用强大的命令操作。您的选择会保存在浏览器中，并在下次访问时持续生效。

更改扩展名和预览

[Screenshot: Extension Badge Placeholder]

点击中央的扩展名徽章即可更改文件格式。这将控制：

  • 语法高亮: 改变颜色以匹配例如 rs 或 js 等编程语言。
  • 预览显示: 按下 Opt/Alt + l 切换预览。md 格式将优雅地呈现，而源代码则显示带有高亮的结构。

3. 网络和自动保存

Leaf 的设计使您永远不必考虑"保存"。

离线/在线自动切换

[Screenshot: Network Status Indicator Placeholder]

左下角的网络指示器显示您当前的状态。

  • 在线时: 您所做的每一个更改都会自动保存到 Google 云端硬盘。
  • 离线时: 如果网络断开，保存功能会立即切换到您浏览器内的一个安全区域 (IndexedDB)。无需停止工作。
  • 重新连接时: 一旦重新上线，Leaf 将在后台执行"静默同步"，自动将您离线时编辑的内容上传到 Google 云端硬盘。
