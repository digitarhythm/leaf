1はじめに

Leafはブラウザで動作するテキストエディタです。インストール不要で、https://leaf.digitarhythm.net/login にアクセスするだけで利用できます。macOS・Windows・Linux向けのデスクトップアプリ版もあります。

Googleログイン vs ゲストモード

Leafには2つの起動方法があります。用途に合わせて選んでください。

🔑 Googleアカウントでログイン

Google Driveと連携して自動保存。複数のデバイスからデータにアクセスできます。カテゴリ（フォルダ）機能を使って整理が可能。

👤 ゲストモードで使う

ログイン不要ですぐに使えます。データはブラウザのローカルストレージに保存されます。ローカルファイルとして書き出しも可能。

ヒント: ゲストモードでも設定画面から「Googleログイン」ボタンでいつでもアカウントへ移行できます。

1Getting Started

Leaf is a browser-based text editor. No installation required — just open https://leaf.digitarhythm.net/login to get started. Desktop apps for macOS, Windows, and Linux are also available.

Login vs Guest Mode

Leaf offers two ways to get started. Choose the one that fits your needs.

🔑 Sign in with Google

Enables Google Drive integration with auto-save. Access your data from any device. Organize sheets with categories (folders).

👤 Use without login

Start immediately with no account required. Data is saved in the browser's local storage. You can also export sheets as local files.

Tip: Even in guest mode, you can switch to a Google account anytime via the "Google Login" button in Settings.

2基本操作

シートを作成する

Leafでは「シート」が1つの文書を表します。複数のシートをタブで管理できます。

  • 1ツールバー左の ＋（新規シート） ボタンをクリックします。
  • 2新しいタブが開き、空のシートが作成されます。
  • 3エディタエリアをクリックして文字を入力できます。
ショートカット: Alt + N で新規シートを素早く作成できます。

保存について

Leafはシートの種類に応じて保存方法が異なります。

  • Googleアカウント（カテゴリあり）: 編集が止まると約2秒後にGoogle Driveへ自動保存されます。
  • Googleアカウント（カテゴリなし）: 自動保存されません。Alt + S で手動保存してください。
  • ゲストモード（自動保存ON）: 編集のたびにブラウザのローカルDBへ保存されます。
  • ゲストモード（自動保存OFF）: Alt + S で手動保存します。
注意: ゲストモードのデータはブラウザのキャッシュと同じ領域に保存されます。ブラウザのデータを消去すると失われる場合があります。大切なデータはローカルファイルとして書き出すか、Googleアカウントでご利用ください。

タブを切り替える

  • 1画面上部のタブバーに、作成したシートが並んで表示されます。
  • 2タブをクリックするとシートを切り替えられます。
  • 3タブをドラッグして順番を変えられます。
  • 4タブ右端の × ボタンでシートを閉じます。
ヒント: タブバーの空き領域をダブルクリックすると新規シートを作成できます。Alt + [ / Alt + ] でタブをキーボードで切り替えられます。

2Basic Usage

Creating Sheets

In Leaf, each document is called a "sheet." You can manage multiple sheets in tabs.

  • 1Click the ＋ (New Sheet) button on the left side of the toolbar.
  • 2A new tab opens with an empty sheet.
  • 3Click the editor area and start typing.
Shortcut: Alt + N creates a new sheet quickly.

Saving Your Work

How sheets are saved depends on the type of sheet.

  • Google Account (with category): Auto-saved to Google Drive ~2 seconds after you stop typing.
  • Google Account (no category): Not auto-saved. Use Alt + S to save manually.
  • Guest Mode (auto-save ON): Saved to the browser's local database on each edit.
  • Guest Mode (auto-save OFF): Use Alt + S to save manually.
Note: Guest mode data is stored in the browser's local storage. It may be lost if you clear browser data. For important content, export as a local file or use a Google account.

Switching Tabs

  • 1All open sheets appear as tabs in the tab bar at the top.
  • 2Click a tab to switch to that sheet.
  • 3Drag tabs to reorder them.
  • 4Click the × button on the right of a tab to close it.
Tip: Double-click an empty area in the tab bar to create a new sheet. Use Alt + [ / Alt + ] to switch tabs with the keyboard.

3ファイル管理

カテゴリ（Google Driveフォルダ）

カテゴリはGoogle Drive内のフォルダに対応しています。シートをカテゴリに割り当てることで、Drive上の特定のフォルダへ自動的に保存されます。

  • 1ツールバーの カテゴリドロップダウン をクリックします。
  • 2一覧からカテゴリを選択するか、新規カテゴリを作成します。
  • 3カテゴリを割り当てると次の保存からDriveの対応フォルダへ保存されます。
ヒント: カテゴリなしのシートは「Untitled」扱いになり、Google Driveには自動保存されません。

Google Driveとの同期

Googleアカウントでログインすると、Leaf専用のフォルダがGoogle Drive内に自動的に作成されます。

  • 編集が一定時間止まると自動的にDriveへ同期します
  • Alt + S でいつでも手動保存できます
  • オフライン中に編集した内容は、次回オンラインになったときに同期されます
  • 別デバイスで編集した場合、次回起動時に最新版を取り込みます

ローカルファイルを開く・保存する

パソコン上のファイルを直接開いて編集することもできます。

  • 1ローカルファイルを開く: ツールバーのフォルダアイコン（Alt + O）をクリックしてファイルを選択します。
  • 2ファイルの内容がエディタに読み込まれ、タブにファイル名が表示されます。
  • 3保存: Alt + S で元のファイルに上書き保存します（File System Access API対応ブラウザ）。
ヒント: ローカルファイルのタブにはファイル名がそのまま表示されます。ゲストモードでは自動保存のON/OFFを設定画面から変更できます。
新規ローカルファイル作成: Alt + Shift + N でファイル名とフォルダを指定して新しいローカルファイルを作成できます。

3File Management

Categories (Google Drive Folders)

Categories correspond to folders in Google Drive. Assigning a category to a sheet will auto-save it to that specific Drive folder.

  • 1Click the Category dropdown in the toolbar.
  • 2Select a category or create a new one.
  • 3The sheet will be saved to the corresponding Drive folder on the next save.
Tip: Sheets without a category are treated as "Untitled" and will not be auto-saved to Google Drive.

Google Drive Sync

When you sign in with Google, Leaf automatically creates a dedicated folder in your Google Drive.

  • Auto-syncs to Drive a few seconds after you stop typing
  • Use Alt + S to save manually at any time
  • Changes made offline are synced the next time you go online
  • Edits made on another device are pulled in on next launch

Opening and Saving Local Files

You can open and edit files directly from your computer.

  • 1Open a file: Click the folder icon in the toolbar (Alt + O) and select a file.
  • 2The file contents load in the editor, and the tab shows the filename.
  • 3Save: Press Alt + S to overwrite the original file (requires a File System Access API–compatible browser).
Tip: Local file tabs display the actual filename. In guest mode, you can toggle auto-save in Settings.
Create a new local file: Alt + Shift + N lets you specify a name and folder for a new local file.

4エディタ機能

Vimモード

LeafのエディタはVimのキーバインドをサポートしています。Vimに慣れているユーザーはそのまま使えます。

  • 設定画面の Vimモード スイッチでON/OFF切り替え（Alt + V でも切り替え可）
  • Normal / Insert / Visual モードに対応
  • Vimモードがオフの場合は通常のテキストエディタとして動作

Markdownプレビュー

拡張子が .md のシートでMarkdownプレビューを表示できます。

  • 1Alt + P または ツールバーのプレビューボタンでプレビューモードに切り替えます。
  • 2通常モードではプレビューがエディタの上に重なって表示されます。
  • 3分割プレビューは Alt + Shift + P で切り替えられます（左:エディタ、右:プレビュー）。
  • 4Esc でエディタに戻ります。
  • Mermaidダイアグラムのリアルタイムレンダリングをサポート
  • コードブロックの構文ハイライト（highlight.js）
  • プレビュー内でカーソルキー・PgUp・PgDn・Home・Endでスクロール可能

構文ハイライト・ファイル形式

タブのファイル形式（拡張子）を変更することで、対応する構文ハイライトが適用されます。

  • 1ツールバー右側の 拡張子ドロップダウン（例: .txt）をクリックします。
  • 2リストから目的の形式を選択します（.js, .py, .rs など100種類以上）。
  • 3エディタのシンタックスハイライトが即座に切り替わります。

4Editor Features

Vim Mode

Leaf's editor supports Vim keybindings. If you're familiar with Vim, you can use it right away.

  • Toggle in Settings under Vim Mode, or press Alt + V
  • Supports Normal, Insert, and Visual modes
  • When disabled, the editor behaves as a standard text editor

Markdown Preview

Sheets with a .md extension support live Markdown preview.

  • 1Press Alt + P or click the preview button in the toolbar.
  • 2In standard mode, the preview overlays the editor.
  • 3Use Alt + Shift + P for split view (editor on left, preview on right).
  • 4Press Esc to return to the editor.
  • Real-time Mermaid diagram rendering
  • Code block syntax highlighting via highlight.js
  • Scroll preview with arrow keys, PgUp, PgDn, Home, End

Syntax Highlighting & File Format

Changing the file extension on a tab applies the corresponding syntax highlighting.

  • 1Click the extension dropdown on the right side of the toolbar (e.g., .txt).
  • 2Select your target format from the list (.js, .py, .rs, and 100+ more).
  • 3Syntax highlighting updates instantly.

6ターミナル デスクトップ版のみ

デスクトップアプリ（Tauri版）にはターミナルが内蔵されています。エディタと並べて使うことができます。

  • 1Alt + T で新しいターミナルタブを開きます。
  • 2ターミナルはタブバーにシートと並んで表示されます。クリックして切り替えられます。
  • 3ターミナルと文書を並べて表示するには、ターミナルタブで Alt + Shift + P を押してください（分割ビュー）。
  • 4ターミナルタブを閉じるにはタブの × ボタンを押します。
ヒント: ターミナルとエディタは独立して動作します。ターミナルを開いていてもエディタのショートカットはそのまま使えます。

6Terminal Desktop only

The desktop app (Tauri) includes a built-in terminal that you can use alongside the editor.

  • 1Press Alt + T to open a new terminal tab.
  • 2The terminal appears in the tab bar alongside your sheets. Click to switch.
  • 3To view the terminal and a document side by side, press Alt + Shift + P in the terminal tab.
  • 4Close a terminal tab by clicking the × button on the tab.
Tip: The terminal runs independently of the editor. Editor shortcuts remain fully functional even with a terminal open.
