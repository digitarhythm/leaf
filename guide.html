2

タブ管理

Tab Management

複数のシートをタブで管理できます。タブには色付きドットが表示され、ひと目で区別できます。

Manage multiple sheets as tabs. Each tab has a colored dot for easy identification.

新規シート作成

New Sheet

ツールバーの「＋」ボタン、またはタブバーの空き領域をダブルクリックで新規シートを作成。Alt+N でも可。

Click the "+" button in the toolbar, or double-click the empty area in the tab bar. Also Alt+N.

タブ切り替え

Switch Tabs

タブをクリックで切り替え。Alt+[ で左、Alt+] で右のタブへ移動できます。

Click a tab to switch. Alt+[ for left, Alt+] for right.

タブを閉じる

Close Tab

タブの「×」ボタン、または Alt+W でアクティブなタブを閉じます。未保存の場合は確認ダイアログが表示されます。

Click the "×" on a tab, or press Alt+W. Unsaved changes will prompt a confirmation dialog.

タブカラー

Tab Colors

各シートには自動的にランダムな色が割り当てられます。タブのドットで視覚的に区別できます。

Each sheet is automatically assigned a random color, shown as a dot on the tab for visual identification.

💡 ヒント： 💡 Tip: Alt+M を押すと、全シートの一覧ダイアログが表示され、プレビューを確認しながらシートを選択できます。 Press Alt+M to open the sheet selection dialog, where you can preview each sheet's content before switching.
3

ツールバーのボタン

Toolbar Buttons

ツールバーには主要な操作ボタンが並んでいます。各ボタンにカーソルを乗せるとツールチップが表示されます。

The toolbar contains all major action buttons. Hover over each button to see its tooltip.

新規シート Alt+N

New Sheet Alt+N

新しい空のシートを作成します。ローカルファイルとして作成するには Alt+Shift+N。

Create a new empty sheet. Use Alt+Shift+N to create a local file.

Google Driveを開く Alt+O

Open from Drive Alt+O

Google Driveに保存されているファイルをカテゴリ別に閲覧・開けます。

Browse and open files stored in Google Drive, organized by category.

ローカルファイルを開く

Open Local File

デバイス上のローカルファイルを開いて編集できます。保存後は元のファイルに上書きされます。

Open local files from your device. Saving overwrites the original file.

カテゴリ（フォルダ）

Category (Folder)

Google Drive上のフォルダに対応。シートをカテゴリで分類して管理できます。

Corresponds to Google Drive folders. Organize sheets by category.

プレビュー Alt+L

Preview Alt+L

Markdownのプレビュー表示に切り替えます。全画面プレビューとスプリット表示を切り替え可能。

Switch to Markdown preview. Toggle between full-screen and split view.

設定 Alt+,

Settings Alt+,

Vimモード・エディタテーマ・フォントサイズ・保存動作などを変更できます。

Configure Vim mode, editor theme, font size, save behavior, and more.

4

テキストエディタ

Text Editor

LeafのエディタはAce Editorをベースに構築されています。多彩なファイル形式と強力な編集機能をサポートします。

Leaf's editor is built on Ace Editor, supporting a wide range of file formats and powerful editing features.

Vimモード

Vim Mode

Vimキーバインドに完全対応。ノーマル・インサート・ビジュアルモードを切り替えながら高速編集できます。設定から無効化可能。

Full Vim keybinding support. Switch between Normal, Insert, and Visual modes. Can be disabled in Settings.

シンタックスハイライト

Syntax Highlighting

Markdown, Rust, JavaScript, TypeScript, Python, C/C++など25種類以上のファイル形式に対応。ファイル拡張子で自動判別。

Supports 25+ file types including Markdown, Rust, JavaScript, TypeScript, Python, C/C++. Auto-detected by extension.

検索・置換 Alt+F

Search & Replace Alt+F

インライン検索ダイアログ。正規表現・大文字小文字区別・全置換に対応。画面中央に表示されます。

Inline search dialog with regex, case-sensitive, and replace-all support. Appears centered on screen.

自動保存

Auto Save

入力が止まると自動的にGoogle Driveへ保存されます。ステータスバーで保存状態を確認できます。

Auto-saves to Google Drive when you stop typing. Check the status bar for save state.

💡 ステータスバー右端の拡張子（例：.md）をクリックすると、ファイル種別を変更できます。シンタックスハイライトがリアルタイムで切り替わります。 Click the file extension in the status bar (e.g., .md) to change the file type. Syntax highlighting updates in real time.

全画面プレビュー Alt+L

Full Preview Alt+L

エディタを隠してプレビューを全画面表示。読み取り専用で内容を確認したい時に便利。

Hide the editor and display the full-screen preview. Great for reading without editing.

スプリットプレビュー Alt+E

Split Preview Alt+E

エディタとプレビューを左右に並べて同時表示。編集しながらリアルタイムでレンダリングを確認できます。

Show the editor and preview side by side. See real-time rendering while you edit.

Mermaidダイアグラム

Mermaid Diagrams

コードブロックに ```mermaid と書くと、フローチャートやシーケンス図を自動描画します。

Write ```mermaid in a code block to automatically render flowcharts and sequence diagrams.

コードハイライト

Code Highlighting

コードブロックのシンタックスハイライトにhighlight.jsを使用。Tokyo Night Darkテーマで表示されます。

Uses highlight.js for code block syntax highlighting with the Tokyo Night Dark theme.

6

Google Drive 同期

Google Drive Sync

ログイン後はすべてのシートが自動的にGoogle Driveに同期されます。どのデバイスからでも同じデータにアクセスできます。

After login, all sheets automatically sync to Google Drive. Access the same data from any device.

1

Googleアカウントでログイン

Sign in with Google

「Googleでサインイン」ボタンをクリックしてGoogleアカウントを認証します。Google Driveへのアクセス権限を許可してください。

Click "Sign in with Google" and authenticate your Google account. Grant access to Google Drive.

2

自動でフォルダ作成

Automatic Folder Creation

初回ログイン時、Google Drive上に「Leaf」フォルダが自動作成されます。すべてのシートはこのフォルダ（またはサブフォルダ）に保存されます。

On first login, a "Leaf" folder is automatically created in Google Drive. All sheets are saved here (or in subfolders).

3

カテゴリでフォルダ管理

Folder Management via Categories

シートをカテゴリ（フォルダ）に分類できます。カテゴリはGoogle Drive上のサブフォルダとして作成されます。ファイルを開くダイアログからカテゴリの作成・削除・名前変更が可能です。

Organize sheets into categories (folders), which become subfolders in Google Drive. Create, delete, and rename categories from the file open dialog.

4

オフラインでも使える

Works Offline Too

インターネット接続がない場合もローカルDBにデータが保持されます。再接続時に自動的にDriveと同期されます。

Data is kept in a local DB even without internet. Automatically syncs with Drive when reconnected.

7

ターミナル デスクトップ版のみ

Terminal Desktop Only

デスクトップ版（macOS・Windows・Linux）ではターミナルをタブとして開けます。エディタとターミナルを同じウィンドウで使えます。

The desktop app (macOS, Windows, Linux) lets you open a terminal as a tab alongside your editor.

新規ターミナル Alt+T

New Terminal Alt+T

新しいターミナルタブを開きます。複数のターミナルを並行して使えます。

Open a new terminal tab. Multiple terminals can run in parallel.

スプリット表示 Alt+L

Split View Alt+L

ターミナルタブでAlt+Lを押すと、右側にMarkdownプレビューを表示するスプリットモードになります。

Press Alt+L in a terminal tab to show a Markdown preview split on the right.

スプリット時の操作

Split Controls

スプリット時にAlt+Wを押すと「どちらを閉じますか？」ダイアログが表示されます。左右キーで選択、Enterで確定。

Press Alt+W in split mode to choose which pane to close. Use arrow keys and Enter to confirm.

フォーカス管理

Focus Management

ダイアログを閉じると自動的にターミナルにフォーカスが戻ります。キーボードで操作を続けられます。

Focus automatically returns to the terminal when dialogs are closed. Continue working with the keyboard.
