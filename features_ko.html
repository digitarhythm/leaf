1. 메인 메뉴 (좌측 상단)

좌측 상단 메뉴 바에는 문서 관리를 위한 3개의 핵심 아이콘이 있습니다.

[Screenshot: Menu Bar Icons Placeholder]
🆕

새 시트 Opt/Alt + n

Google 드라이브에 새 텍스트 파일을 만들어 즉시 작성을 시작하세요. 파일명은 걱정하지 마세요. Leaf가 알아서 "제목 없음"으로 관리합니다.

🗂️

시트 선택 대화상자 Opt/Alt + m

Leaf의 "관리 화면". 범주별로 정리된 문서를 미리 보기와 함께 탐색할 수 있습니다. 파일명이 아닌 "내용"으로 판단하는 Leaf만의 독특한 방식입니다.

📂

로컬 파일 열기 Opt/Alt + o

PC의 기존 파일을 엽니다. 원본 파일의 인코딩(예: BOM)이 유지됩니다. 범주를 변경하여 Google 드라이브로 옮기면 자동으로 클라우드 동기화가 설정됩니다.

2. 작성 지원 기능 (상태 표시줄)

하단 상태 표시줄에서 실시간으로 편집기를 맞춤 설정할 수 있습니다.

Vim 모드 전환

[Screenshot: Vim Toggle Button Placeholder]

우측의 "Vim" 버튼으로 전문가용 Vim 단축키를 활성화하세요. H/J/K/L로 이동하고 강력한 명령어를 브라우저에서 바로 사용하세요. 설정은 저장되어 다음 방문 시에도 유지됩니다.

확장자 확인 및 미리보기

[Screenshot: Extension Badge Placeholder]

중앙의 확장자 배지를 클릭하여 파일 포맷을 변경하세요. 이로써 다음을 제어할 수 있습니다:

  • 구문 강조: rs 또는 js처럼 언어에 맞게 색상이 변경됩니다.
  • 미리보기: Opt/Alt + l 키를 눌러 미리보기를 전환하세요. md 포맷은 미려하게 렌더링되며, 소스 코드는 강조 표시가 유지됩니다.

3. 네트워크 및 자동 저장

Leaf에서는 "저장"을 신경 쓸 필요가 없습니다.

자동 오프라인/온라인 전환

[Screenshot: Network Status Indicator Placeholder]

좌측 하단 인디케이터에 네트워크 상태가 표시됩니다.

  • 온라인 시: 작성한 내용이 Google 드라이브에 자동으로 저장됩니다.
  • 오프라인 시: 연결이 끊기면 즉시 브라우저의 안전한 영역(IndexedDB)으로 저장이 전환됩니다. 작업을 멈추지 마세요.
  • 재연결 시: 오프라인 동안 반영되지 않은 내용을 "백그라운드 동기화"로 Google 드라이브에 안전하게 자동 업로드합니다.
