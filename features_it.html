Guida alle funzionalità

Automatizza la "Gestione" per concentrarti sulla "Scrittura"

1. Menu principale (In alto a sinistra)

La barra dei menu contiene tre icone chiave per gestire il ciclo di vita del tuo documento.

[Screenshot: Menu Bar Icons Placeholder]
🆕

Nuovo foglio Opt/Alt + n

Inizia a scrivere immediatamente creando un nuovo file su Google Drive. Leaf lo gestisce automaticamente come "Senza titolo".

🗂️

Selezione del foglio Opt/Alt + m

La schermata "Gestione". Sfoglia i tuoi documenti per categoria con l'anteprima del contenuto.

📂

Carica file locale Opt/Alt + o

Apri file esistenti sul tuo PC. Cambia la loro categoria per abilitare la sincronizzazione nel cloud.

2. Supporto alla scrittura (Barra di stato)

Personalizza il comportamento dell'editor in tempo reale con la barra in basso.

Modalità Vim

[Screenshot: Vim Toggle Button Placeholder]

Il pulsante "Vim" abilita le scorciatoie avanzate (H/J/K/L) nel browser.

Estensione e Anteprima

[Screenshot: Extension Badge Placeholder]

Fai clic sul badge per cambiare il formato. Questo controlla:

  • Evidenziazione della sintassi: Modalità per rs o js.
  • Anteprima: Opt/Alt + l alterna l'anteprima, ideale per formati md e codice.

3. Rete e salvataggio automatico

Leaf salva in background senza interruzioni.

Offline/Online

[Screenshot: Network Status Indicator Placeholder]

L'indicatore di rete mostra lo stato attuale.

  • Online: Salva automaticamente su Google Drive.
  • Offline: Il salvataggio sfrutta il tuo browser in modo sicuro (IndexedDB).
  • Riconnesso: Leaf carica nel cloud le bozze automaticamente.
